## 1. El Componente Hueco

Puedes rellenar valores de forma aislada para resolver preguntas clásicas.

<Hueco 
  id="hueco_bloque" 
  pregunta="¿Cuál es la raíz cuadrada de 144?" 
  correct="12" 
  pista="Es una docena de docenas..." 
/>

Pero también, el componente `<Hueco>` es extremadamente potente para usarlo **inline**. Por ejemplo, podemos decir que la respuesta de $5 \times 5$ es <Hueco id="hueco_inline" correct="25" /> directamente en el texto sin romper el flujo de lectura. Y si quiero calcular $7 + 3$, sé que el resultado es <Hueco id="hueco_inline_2" correct="10" />.

## 2. Preguntas Multi-Opción (Tarjetas Elegantes)

A continuación, una pregunta clásica. Fíjate en cómo cambian las tarjetas y aparece el sello (✓ o ✗) al responder.

<Pregunta
  id="pregunta_1"
  texto="¿Quién es considerado el 'Príncipe de las Matemáticas'?"
  correct="gauss"
  opciones={[
    { value: "euler", texto: "Leonhard Euler" },
    { value: "gauss", texto: "Carl Friedrich Gauss" },
    { value: "newton", texto: "Isaac Newton" }
  ]}
/>

## 5. Álgebra Lineal: Matrices Interactivas

Añade los valores que faltan para formar la **Matriz Identidad** de $3 \times 3$. Fíjate en los corchetes clásicos.

<MatrizInteractiva 
  id="matriz_identidad"
  pregunta="Rellena las celdas para completar la matriz identidad:"
  correct={[
    ["1", "0", "0"],
    ["0", "1", "0"],
    ["0", "0", "1"]
  ]}
/>

## 6. Variables Dinámicas (Deslizadores)

Puedes inyectar interactividad real en el texto usando un deslizador. Cambia el valor de $X$ a continuación y fíjate cómo se actualiza la frase matemática de abajo en tiempo real.

> Modifica la variable $X$: <DeslizadorEnLine name="x" min={1} max={10} initial={5} label="Valor de X =" />

Si multiplicamos nuestro valor por dos, el resultado de $2X$ es igual a: <DynamicValue name="x" fallback={5} /> $\times 2$. *(La simulación calcularía esto reactivamente)*.

## 7. Clasificador Analítico (Cubos Lógicos)

Una alternativa al drag & drop lineal es el clasificador. Clasifica las siguientes ecuaciones según su tipo.

<Clasificador 
  id="clasificador_1"
  pregunta="Clasifica las ecuaciones en Lineales o Cuadráticas:"
  buckets={[
    { id: "lineal", title: "Lineales (Grado 1)" },
    { id: "cuadratica", title: "Cuadráticas (Grado 2)" },
    { id: "cubica", title: "Cúbicas (Grado 3)" },
  ]}
  items={[
    { id: "i1", content: "$2x + 5 = 0$", bucketId: "lineal" },
    { id: "i2", content: "$x^2 - 4 = 0$", bucketId: "cuadratica" },
    { id: "i3", content: "$-x = 7$", bucketId: "lineal" },
    { id: "i4", content: "$3x^2 + 2x - 1 = 0$", bucketId: "cuadratica" },
    { id: "i5", content: "$x^3 + 2x^2 + x - 1 = 0$", bucketId: "cubica" },
    { id: "i6", content: "$4x^3 - 2x^2 + 5x - 10 = 0$", bucketId: "cubica" },
  ]}
/>
