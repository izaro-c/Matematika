# Gobierno multi-IA

Esta carpeta define cómo se toman, registran y transfieren decisiones entre herramientas de IA. Es neutral respecto al proveedor y cambia con menor frecuencia que la capa operativa `ai/`.

## Alcance

- autoridad, responsabilidades y resolución de conflictos;
- protocolo de ejecución, revisión y relevo;
- criterios para cambiar reglas comunes o adaptadores.

El protocolo formal está en [`protocol.md`](protocol.md). Las instrucciones globales están en [`../../AGENTS.md`](../../AGENTS.md) y el trabajo diario en [`../../ai/README.md`](../../ai/README.md).

No se guardan aquí prompts de una fase, estados temporales, informes de ejecución ni configuración específica de una herramienta.
