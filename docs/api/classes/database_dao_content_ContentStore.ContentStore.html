    Class ContentStore

    Singleton ContentStore

    Centraliza y gestiona la base de datos de contenido MDX de Matematika. Lee, valida y compila los módulos a través de Vite en tiempo de ejecución.

    Remarks

    Esta clase actúa como un Singleton en la aplicación.
