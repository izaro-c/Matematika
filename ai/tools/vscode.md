# VS Code

## Papel recomendado

Puesto de control humano para inspeccionar diffs, ejecutar comandos, comparar propuestas y aprobar cambios de asistentes integrados.

## Uso

- Abrir el repositorio en su raíz para que los agentes encuentren `AGENTS.md`.
- Revisar el diff antes y después de cada relevo.
- Ejecutar validaciones en terminal y conservar sus resultados resumidos.
- Mantener ajustes personales fuera del gobierno común salvo que deban compartirse.

Las extensiones de IA siguen la misma jerarquía; `.vscode/` no debe convertirse en una fuente paralela de reglas.
